# Mobile E-Commerce Backend - Requirements

## Java Dependencies (Gradle)

### Core Framework
- org.springframework.boot:spring-boot-starter-web (Web & REST API)
- org.springframework.boot:spring-boot-starter-data-jpa (Database ORM)

### Database
- org.postgresql:postgresql (PostgreSQL Driver)

### Email Service
- org.springframework.boot:spring-boot-starter-mail (SMTP Mail)

### API Documentation
- org.springdoc:springdoc-openapi-starter-webmvc-ui:2.0.2 (Swagger/OpenAPI)

### Code Generation
- org.projectlombok:lombok (Reduce Boilerplate)

### Testing
- org.springframework.boot:spring-boot-starter-test (JUnit)

### Environment Variables
- io.github.cdimascio:dotenv-java (Tải .env files)

## System Requirements

### Software
- JDK 21+
- PostgreSQL 12+
- Git
- Gradle (Auto-downloaded via Wrapper)

### Runtime
- RAM: 2GB minimum
- Disk: 1GB for dependencies & build artifacts

## Development Tools (Optional)
- VS Code with Java Extension Pack
- IntelliJ IDEA Community Edition
- Postman (API Testing)

## Database Setup
```sql
CREATE DATABASE Mobile_ecommerce;
```

## Server Configuration
- Port: 8000 (default, configurable via .env)
- Context Path: /api
